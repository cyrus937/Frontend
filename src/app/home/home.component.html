Bienvenue sur la page d'envoi de SMS, Web SMS

Web SMS est une plateforme permetant d’envoyer des SMS en masse sur tous les réseaux exercant sur le teritoire camerounais.

Vous pouvez créer des groupes de discussions ou alors envoyer des messages individuels Il sera aussi possible de consulter les performances de transmission et le solde de crédit restant lorsque ceci est souhaité.
